# Pickleball Analytics Project

This project analyzes data from a pickleball academy to uncover insights into player performance, training sessions, match outcomes, and court usage.

## Scenario
The academy wants to understand:
- Who are the top performing players
- Which courts are most used
- What factors impact winning

## Datasets
- players.csv: player profiles
- sessions.csv: training sessions
- matches.csv: match results
- courts.csv: court usage

## Tools
Python, SQL, Power BI, Tableau
